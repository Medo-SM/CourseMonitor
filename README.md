# CourseMonitor: University Monitoring System

## About the Project

This is a web application designed to help university instructors monitor student attendance and academic performance. The system is built for use by lecturers, department heads, and administrators.

## Features

*   **Course Management:** Add new courses and assign lecturers.
*   **Student Management:** Register students and organize them into groups.
*   **Attendance Tracking:** Mark student attendance (Present, Absent, Late).
*   **Grade Management:** Input grades for coursework, quizzes, and exams.
*   **Report Generation:** Generate reports on student performance and attendance.

## Technologies Used

*   **Backend:** PHP
*   **Frontend:** HTML, CSS, JavaScript (Bootstrap 5)
*   **Database:** MySQL

## Installation

1.  Install **XAMPP** or a similar web server environment (Apache, MySQL, PHP).
2.  Clone or download this repository and place it in your web server's document root (e.g., `htdocs` for XAMPP).
3.  Create a new MySQL database named `CourseMonitor` using `phpMyAdmin` or a similar tool.
4.  Import the `CourseMonitor.sql` file into the newly created database.
5.  Verify your database connection settings (username, password) in `config/db.php`.
